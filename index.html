Stroke Prediction Project

Predicting Stroke Risk Using Machine Learning

Why This Matters

Stroke is the second leading cause of death globally, responsible for about 11% of all deaths. In this project, we apply machine learning models to predict the likelihood of stroke based on features such as age, hypertension, heart disease, glucose level, BMI, and smoking status.

Dataset Features

  • Demographics: Age, Gender, Marital Status
  • Medical: Hypertension, Heart Disease, BMI, Avg Glucose
  • Lifestyle: Work Type, Smoking Status, Residence Type
  • Target Variable: Stroke (0 = No, 1 = Yes)

Machine Learning Models Used

  • Logistic Regression
  • Decision Tree
  • Random Forest
  • Support Vector Machine (SVM)
  • K-Nearest Neighbors (KNN)
